请生成一个完整可运行的单文件 HTML 页面，实现一个“随机抽奖模拟开奖网页”。

这是一个娱乐性质的随机抽奖模拟器，玩法参考常见号码球开奖形式：从 1-49 中随机抽取 6 个普通号码和 1 个特别号码。请注意：不要加入任何真实彩票购买、预测、投注、赔率、充值、赌博相关功能。

具体要求：

【核心功能】
1. 从 1 到 49 中随机抽取号码。
2. 每次开奖生成 6 个普通号码 + 1 个特别号码。
3. 同一次开奖的 7 个号码不能重复。
4. 普通号码按从小到大排序显示，特别号码单独显示。
5. 点击“开始开奖”后，按钮进入禁用状态。
6. 开奖完成后按钮恢复可点击。
7. 支持再次开奖。
8. 显示最近 5 次开奖历史，每条历史包含：
   - 开奖时间
   - 6 个普通号码
   - 1 个特别号码

【动画要求】
1. 点击开始后，页面先显示一个“滚动中的号码球”动画。
2. 号码不要一次性全部显示，应按顺序逐个揭晓。
3. 每个号码出现时有弹跳、放大、发光动画。
4. 特别号码最后揭晓，并有更明显的金色发光效果。
5. 开奖完成后出现轻微庆祝动画，例如彩带、闪光、背景光效等，可用 CSS 实现。

【页面设计】
1. 使用现代化 UI 风格。
2. 背景使用渐变色。
3. 主体区域使用卡片布局。
4. 号码球为圆形，带阴影。
5. 普通号码根据号码范围分成不同颜色：
   - 1-16：红色
   - 17-33：蓝色
   - 34-49：绿色
6. 特别号码使用金色或紫色。
7. 页面在手机端也要美观，使用响应式布局。
8. 页面底部显示免责声明：
   “本页面仅用于娱乐和随机抽奖演示，不涉及真实彩票、投注或金钱交易。”

【技术要求】
1. 只使用 HTML、CSS、JavaScript。
2. 所有代码写在一个 HTML 文件中。
3. 不使用任何外部库或框架。
4. 不访问网络接口。
5. 随机逻辑使用 JavaScript 实现。
6. 代码需要有适量注释，方便理解和修改。

【输出要求】
请直接输出完整 HTML 文件代码。
不要输出解释说明。
不要省略任何代码。
